## What AeroAI Does

AeroAI supports aerodynamic concept work for rail transport, with its main emphasis on bullet train nose geometry. By applying predictive modeling, it helps users compare forms that may lessen drag and better match the demands of high-speed service.

It is a practical fit for researchers, engineers, and design teams that need an early-stage tool for transport studies, simulation pipelines, or concept screening. Rather than serving as a substitute for engineering validation, it provides a focused way to narrow design options and identify promising directions.

## Using AeroAI

AeroAI is meant for evaluating aerodynamic concepts, not for broad-purpose modeling tasks.

Typical workflow:
1. Open the web interface.
2. Provide or select the train design context.
3. Review the predicted nose design output.
4. Compare the result against drag, efficiency, and performance goals.
5. Iterate as needed to explore different design directions.

Example local preview:
`python -m http.server 8000`

Then open:
`http://localhost:8000`
